이 문서는 프로젝트의 모든 유즈케이스(UC)를 포함한 완전한 제품 백로그입니다. 각 기능별 **전체 우선순위(Global Priority)**를 포함하여 개발 로드맵 수립을 지원합니다.

- **P1 (최우선):** 서비스 출시에 필수적인 핵심 기능 (MVP)
- **P2 (우선):** 서비스 안정화 및 기본 사용자 경험을 위한 기능
- **P3 (보통):** 부가 가치를 제공하는 확장 기능
- **P4 (나중):** 추후 고도화 단계에서 고려할 기능
